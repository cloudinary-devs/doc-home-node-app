## Docs Home Node Sample

The getting started sample uploads remote image to Cloudinary and generates URLs for applying various image transformations using addons on the uploaded files.

### Setting up
1. Make sure you have a Cloudinary account or [register for free](https://cloudinary.com/users/register_free).
2. Before running the sample, copy your `CLOUDINARY_URL` environment variable and replace it in the `.env` file. You can find your credentials in the Cloudinary Console Dashboard.
2. Make sure you're registered for the [AWS Auto-Tagging add-on](https://cloudinary.com/documentation/google_auto_tagging_addon)  (free tier available).
3. Make sure you're registered for the [AWS Auto-Tagging add-on](https://cloudinary.com/documentation/google_auto_tagging_addon) (free tier available).
4. In the `doc-home-node-app` directory, run `npm install` to get all the required modules.
5. Run the demo app using `npm start`.
